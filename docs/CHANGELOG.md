# Changelog

このファイルは、更新履歴を記録するテンプレートです。

「何を変えたか」「何を確認したか」「何がまだ未確認か」を書いてください。

## YYYY-MM-DD

記入例:

### 変更内容

- READMEとdocsを作成
- 初期画面を追加
- 保存処理を追加

### 確認したこと

- アプリが開くことを確認
- 保存ボタンでデータが保存されることを確認
- 再読み込み後もデータが残ることを確認

### 未確認のこと

- 大量データでの動作速度は未確認
- Macアプリ化は未確認

### 次にやること

1. 編集機能を追加する
2. 削除機能を追加する
3. 既知の不具合を確認する

## ルール

- 未確認のものを確認済みとして書かない
- 変更内容だけでなく確認内容も書く
- 次にやることを書く
- ChatGPTやCodexがこのファイルだけ見ても流れが分かるように書く
