# Handoff

このファイルは、次に作業する人・ChatGPT・Codexへ引き継ぐためのテンプレートです。

会話履歴がなくても、このファイルを読めば作業を再開できるように書いてください。

## 現在の目的

記入例:

```text
タスク管理アプリの最初の実用版を作る。
今は保存機能の安定化を優先している。
```

## 現在の状態

記入例:

```text
一覧表示と追加フォームは実装済み。
保存処理は実装済みだが、再読み込み後の復元は未確認。
```

## 直近で行った作業

記入例:

- READMEとdocsを整備した
- タスク追加フォームを作った
- 保存ボタンを追加した

## 次にやること

必ず書いてください。

記入例:

1. 再読み込み後にデータが復元されるか確認する
2. 復元されない場合は保存処理を修正する
3. 修正後に `CURRENT_STATUS.md` と `KNOWN_ISSUES.md` を更新する

## 注意点

記入例:

- 未確認のことを「完了」と書かない
- 新機能を足す前に、保存処理の確認を終える
- 画面の見た目より、まずデータが消えないことを優先する

## 関係ファイル

記入例:

| ファイル | 役割 |
| --- | --- |
| `src/main.js` | 画面と保存処理 |
| `README.md` | プロジェクト入口 |
| `docs/CURRENT_STATUS.md` | 現在の状態 |
| `docs/KNOWN_ISSUES.md` | 不具合一覧 |

## ChatGPT / Codex への次回依頼文

記入例:

```text
README.md と docs/ 以下を読んでから作業してください。
現在は保存処理の確認が最優先です。
新機能は追加せず、再読み込み後にデータが残るかを確認し、
問題があれば修正してください。
修正後は CURRENT_STATUS.md、KNOWN_ISSUES.md、CHANGELOG.md、HANDOFF.md を更新してください。
```
